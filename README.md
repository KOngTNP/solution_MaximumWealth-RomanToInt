## You can add more test case in class MyTest

Example for MaximumWealth
```python
self.assertEqual(myCal.maximumWealth_basic([[1,2,3],[3,6,1]]), 10)
```
Example for RomanToInt
```python
self.assertEqual(myCal.romanToInt("XXVIII"), 28)
```
